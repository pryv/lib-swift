## Motivation

Provide a use case example for the `lib-swift` framework.

## Proposition

We propose **an app** that shows how to :  
* load the package   
* create an event  
* show the last 20 events  
* save the credentials in Apple keychain
  
## Deliverables

### API

- Button to connect to the user's Pryv account
- Example application
